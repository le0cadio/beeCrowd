# Beecrowd Solutions Repository

This repository contains solutions to coding exercises from the Beecrowd platform. The solutions are implemented in Kotlin.

## Structure

- `src/main/kotlin/`: Contains the main Kotlin source files with solutions.
- `.gitignore`: Specifies files and directories to be ignored by Git.

## Getting Started

1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```
2. Open the project in IntelliJ IDEA.
3. Run the solutions from the `src/main/kotlin/` directory.
